## Role Setup

### Default Roles

| Role | Color | Permissions | Auto-Assign |
|------|-------|-------------|-------------|
| @everyone | Default | Basic access | All members |
| Bot | Gray | Bot permissions | Bots only |

### User Roles

| Role | Color | Permissions | How to Get |
|------|-------|-------------|------------|
| **Beta Tester** | Blue | Access to beta channels | Verify email |
| **Contributor** | Green | Access to dev channels | Active feedback |
| **Moderator** | Orange | Moderate channels | Team appointment |
| **Developer** | Red | Full access | Dev team |

### Special Roles

| Role | Color | Purpose |
|------|-------|---------|
| **Founder** | Gold | Early beta testers (first 50) |
| **Bug Hunter** | Purple | Reported critical bugs |
| **Helper** | Cyan | Active in support |

## Verification Flow

### Email Verification

1. User joins Discord
2. Bot DMs verification link
3. User clicks link (confirms email)
4. Bot assigns "Beta Tester" role
5. User gets full server access

### Commands

```
!verify - Start verification process
!help - Show help message
!support - Get support contact
!feedback - Submit feedback
```

## Guidelines

### Beta Program Rules

1. **Be Respectful**
   - Treat all members with respect
   - No harassment or discrimination
   - Keep discussions constructive

2. **Feedback is Valuable**
   - Report bugs constructively
   - Suggest features politely
   - Rate your conversions

3. **Confidentiality**
   - Don't share beta features publicly
   - Respect NDA if applicable
   - Ask before streaming beta features

4. **Support Etiquette**
   - Search before asking
   - Provide details in bug reports
   - Help others when you can

5. **Have Fun!**
   - Share your creations
   - Help the community grow
   - Enjoy converting mods!

### Enforcement

**Warnings:**
1. Verbal warning (DM)
2. Public warning
3. Temporary mute (24 hours)
4. Temporary ban (7 days)
5. Permanent ban

**Immediate Ban Offenses:**
- Harassment or hate speech
- Sharing confidential information
- Spam or advertising
- Impersonating staff
