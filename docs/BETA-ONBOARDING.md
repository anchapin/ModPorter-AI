## What is ModPorter AI?

ModPorter AI is an AI-powered platform that automatically converts Minecraft Java Edition mods to Bedrock Edition add-ons. Our goal is to help mod creators reach Bedrock's 3x larger player base without manual code rewriting.

## Converting Your First Mod

### Upload Process

1. **Login** to your account at https://modporter.ai
2. **Click "New Conversion"** on the dashboard
3. **Upload your mod file** (.jar or .zip)
   - Drag and drop or click to browse
   - Wait for file validation
4. **Configure conversion options** (optional)
   - Target Bedrock version (default: latest)
   - Conversion assumptions (conservative/balanced/aggressive)
5. **Click "Start Conversion"**
   - You'll see a progress indicator
   - Typical conversion time: 2-10 minutes

### Understanding Progress

**Conversion Stages:**
```
0%   - Queued
10%  - Analyzing Java code
40%  - Translating to Bedrock
80%  - Validating output
90%  - Packaging .mcaddon
100% - Complete!
```

### Downloading Results

Once conversion is complete:

1. **Download .mcaddon file**
   - Click the download button
   - File will be available for 30 days
2. **Review conversion report**
   - Success rate
   - Applied assumptions
   - Known issues
3. **Provide feedback**
   - Rate the conversion
   - Report any issues
   - Suggest improvements

## Installing Your Converted Add-on

### Windows 10/11

1. Download the .mcaddon file
2. Double-click the file
3. Minecraft Bedrock will open and import the add-on
4. Create a new world
5. Enable the add-on in world settings:
   - Go to "Resource Packs" → "My Packs" → Activate
   - Go to "Behavior Packs" → "My Packs" → Activate
6. Play your world!

### Xbox

1. Transfer .mcaddon file to Xbox (USB or network share)
2. Open the file from Xbox file explorer
3. Minecraft will import the add-on
4. Enable in world settings

### Mobile (iOS/Android)

1. Download .mcaddon file to device
2. Tap the file to open in Minecraft
3. Add-on will be imported
4. Enable in world settings

## Troubleshooting

### Common Issues

**"Invalid mod file"**
- Ensure file is .jar or .zip format
- Check file size (must be under 100MB)
- Verify mod is for Minecraft Java Edition

**"Conversion failed"**
- Check your internet connection
- Try again in a few minutes
- Contact support if issue persists

**"Add-on not working in Minecraft"**
- Ensure you enabled both Resource and Behavior packs
- Check Minecraft Bedrock version compatibility
- Review conversion report for known issues

**"Missing textures"**
- Some texture formats may not convert perfectly
- Check the conversion report for texture issues
- Manually fix textures if needed (we'll provide guidance)

## Tips for Best Results

### Mod Preparation

**Before Converting:**
1. **Test your Java mod** - Ensure it works in Java Edition
2. **Simplify complex features** - Remove unsupported features if possible
3. **Organize your code** - Well-structured mods convert better
4. **Document custom features** - Helps us understand what you're trying to achieve

### Conversion Settings

**Assumption Levels:**
- **Conservative**: Only convert features with direct Bedrock equivalents
- **Balanced** (recommended): Convert with reasonable assumptions
- **Aggressive**: Attempt conversion of complex features (may require manual fixes)

### Post-Conversion

**After Download:**
1. **Test in Minecraft** - Create a test world
2. **Check all features** - Verify items, blocks, entities work
3. **Review the report** - Note any known issues
4. **Provide feedback** - Help us improve!

## Frequently Asked Questions

**Q: Is my mod safe?**
A: Yes! We only read your mod file, never modify the original. Your converted add-on is yours to keep.

**Q: How long are conversions stored?**
A: Converted files are stored for 30 days. Download and backup your add-ons.

**Q: Can I convert any mod?**
A: Most simple to moderate mods convert well. Complex mods with custom rendering or networking may not work.

**Q: Will converted mods work on all Bedrock platforms?**
A: Yes! Converted add-ons work on Windows 10/11, Xbox, PlayStation, Switch, and mobile.

**Q: Can I sell converted add-ons?**
A: You retain rights to your converted add-ons. Follow Minecraft Marketplace policies for selling.

**Q: How do I leave the beta program?**
A: You can delete your account anytime in account settings. We'd love to hear your feedback first!
