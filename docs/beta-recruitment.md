# Beta User Recruitment Strategy

**Issue:** #1166
**Milestone:** M5: Beta Launch
**Goal:** Recruit 25-50 mod creators, first 5 conversions within week 1

---

## Overview

This document outlines the recruitment strategy for beta testers for Portkit, an AI-powered Minecraft Java to Bedrock mod converter. The target audience is active mod creators who will receive free access in exchange for feedback and bug reports.

## Target Audience

**Primary:** Active Minecraft Java mod creators with published mods
**Secondary:** Modding community leaders and educators
**Tertiary:** Bedrock creators interested in porting existing mods

## Priority Channels

### 1. r/feedthebeast
- **Reach:** 400K+ members, most active Java modding community
- **Approach:** Direct post with beta application link
- **Timing:** Tuesday-Thursday, 6-9 AM PST
- **Guidelines:** Must disclose affiliation, no direct promotion

### 2. Modrinth Discord
- **Reach:** Active mod developers, many already cross-platform aware
- **Approach:** Partner with mods channel moderators, brief announcement
- **Timing:** During server low-traffic hours
- **Key Contacts:** Moderators, plugin API developers

### 3. CurseForge Forums
- **Reach:** Overlapping creator community
- **Approach:** Forum post in Java modding section
- **Timing:** Weekdays, high engagement periods

### 4. Minecraft Marketplace Discord
- **Reach:** Professional creators, potential high-quality testers
- **Approach:** DM to known creators, brief server announcement
- **Note:** More formal, professional tone required

### 5. r/MinecraftBedrock
- **Reach:** Bedrock community interested in content from Java side
- **Approach:** Cross-community bridge post
- **Note:** Emphasize reaching 3x larger Bedrock player base

### 6. Twitter
- **Reach:** Broader Minecraft community, developer visibility
- **Approach:** Thread format with clear call-to-action
- **Timing:** Weekdays, conference-attendance timing

## Beta Offer

| Benefit | Details |
|---------|---------|
| Free Creator tier | Unlimited conversions during beta |
| Direct bug reports | Private Discord channel to dev team |
| Recognition | Named credits in launch announcement |
| Timeline | 8-week beta program |

## Key Messaging

**Primary message:** "conversion accelerator"
- Do: Emphasize 60-80% automated conversion
- Do: Position as time-saver for cross-platform expansion
- Don't: Claim 100% automation (not accurate)

**Value proposition:**
1. Save time converting mods to Bedrock
2. Access Bedrock's 3x larger player base
3. Shape the future of cross-platform modding
4. Free access during beta
