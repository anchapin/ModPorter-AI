# ModPorter AI - Quick Start Guide

**Convert your first Java mod to Bedrock in 5 minutes!**

---

## Prerequisites

- [ ] Minecraft Java Edition mod (.jar or .zip file)
- [ ] Minecraft Bedrock Edition (any platform)
- [ ] ModPorter AI account (sign up at https://modporter.ai)

---

## Step-by-Step Conversion

### 1. Login (1 minute)

```
1. Visit https://modporter.ai
2. Click "Login"
3. Enter your credentials
4. You're in!
```

### 2. Upload Mod (1 minute)

```
1. Click "New Conversion"
2. Drag your .jar/.zip file or click to browse
3. Wait for validation (green checkmark)
```

### 3. Start Conversion (30 seconds)

```
1. (Optional) Adjust settings:
   - Target version: 1.20.0 (default)
   - Assumptions: Balanced (recommended)
2. Click "Start Conversion"
3. Watch the progress!
```

### 4. Download Result (1 minute)

```
1. When complete, click "Download .mcaddon"
2. Save to your computer
3. Review the conversion report
```

### 5. Install in Minecraft (2 minutes)

**Windows:**
```
1. Double-click the .mcaddon file
2. Minecraft opens and imports
3. Create new world
4. Enable add-ons in world settings
5. Play!
```

**Xbox/Mobile:**
```
1. Transfer file to device
2. Open file to import
3. Enable in world settings
4. Play!
```

## Expected Results

### ✅ Converts Well

- Simple items (swords, tools, armor)
- Basic blocks (decorative, ores)
- Simple entities (passive mobs)
- Crafting recipes

### ⚠️ May Need Work

- Complex entities (bosses, custom AI)
- Custom GUIs
- Complex multi-block structures

### ❌ Doesn't Convert

- Custom rendering (shaders, models)
- Network packets
- Java-specific APIs
