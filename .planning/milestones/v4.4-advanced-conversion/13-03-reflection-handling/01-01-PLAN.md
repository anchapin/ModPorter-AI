  <name>Task 1: Create ReflectionDetector utility</name>
  <files>ai-engine/utils/reflection_detector.py</files>
  <action>
Create ReflectionDetector class that:
- Uses javalang to parse Java source
- Detects MethodInvocation nodes for reflection methods:
  - Class.forName(), Class.getName(), Class.getSimpleName()
  - getDeclaredFields(), getFields(), getDeclaredMethods(), getMethods()
  - getDeclaredConstructors(), getConstructors()
  - Field.get(), Field.set(), Field.getInt(), Field.setInt()
  - Method.invoke(), Constructor.newInstance()
  - getAnnotation(), getAnnotations()
- Creates dataclasses:
  - ReflectionCall: method name, target, arguments
  - DynamicClassLoad: class name being loaded
  - FieldAccess: field name, read/write
  - MethodInvocation: method name, target object, arguments
  - AnnotationAccess: annotation type, target
- Implements detect_from_source() method returning list of ReflectionCall
  </action>

  <done>ReflectionDetector imports and basic structure works</done>
</task>

<task type="auto">
  <name>Task 2: Create ReflectionMapper utility</name>
  <files>ai-engine/utils/reflection_mapper.py</files>
  <action>
Create ReflectionMapper class that:
- Takes ReflectionCall objects from detector
- Categorizes patterns:
  - CONVERTIBLE: Can be converted to direct JavaScript
  - WARNING: Needs user attention
  - UNSUPPORTED: No JavaScript equivalent
- Converts safe patterns:
  - Class.getSimpleName() → class.name
  - getDeclaredFields() → static analysis results
  - getMethods() → Object.getOwnPropertyNames()
  - Field.get() → direct property access
- Warns about:
  - Class.forName() dynamic loading
  - Method.invoke() dynamic calls
  - Constructor.newInstance()
  - setAccessible() calls
- Creates ReflectionWarning dataclass with:
  - pattern: The reflection pattern
  - severity: HIGH, MEDIUM, LOW
  - message: Human-readable explanation
  - suggestion: Recommended alternative
- Implements map_reflection() returning ConversionResult with converted code and warnings
  </action>
  <verify>python -c "from ai_engine.utils.reflection_mapper import ReflectionMapper; print('Import OK')"</verify>
  <done>ReflectionMapper imports and basic structure works</done>
</task>
