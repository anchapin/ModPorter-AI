<objective>
Implement potion and effect conversion from Java mods to Bedrock, including status effects, potions, and mob effects.
</objective>

<context>
@ai-engine/converters/advancement_converter.py (recently created converter)
@ai-engine/knowledge/patterns/ (existing pattern library)

# Bedrock Effect System
Bedrock uses:
- minecraft:entity_effects component
- minecraft:damage_over_time
- Effect definitions in entity JSON
- Potion items with custom effects

# Java Effect System
Java mods define:
- MobEffect/MobEffectInstance
- PotionType registry
- Effect amplifiers and durations
- Custom status effects
</context>
