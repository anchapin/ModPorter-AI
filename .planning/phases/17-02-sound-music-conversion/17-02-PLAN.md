<objective>
Implement sound and music system conversion from Java mods to Bedrock, including sound events, music discs, ambient sounds, and custom audio.
</objective>

<context>
@ai-engine/agents/entity_converter.py (recently enhanced)
@ai-engine/knowledge/patterns/ (existing pattern library)

# Bedrock Sound System
Bedrock uses:
- sounds.json for sound event definitions
- audio files in sounds/ directory
- music gamems/ for background music
- Custom sound events via minecraft:sound

# Java Sound System
Java mods use:
- SoundEvent registry entries
- SoundCategory (AMBIENT, BLOCKS, CREATIVE, HOSTILE, ITEM, MASTER, MUSIC, NEUTRAL, PLAYERS, VOICE)
- Custom SoundSource classes
- SoundPool for repeated sounds
</context>
