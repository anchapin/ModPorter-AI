# Bedrock Item System
Bedrock uses:
- Item components in item JSON
- minecraft:item components (damage, durability,enchantments)
- Armor components (armor, toughness, knockback)
- Weapon components (damage, attack_speed)

# Java Item System
Java mods define:
- Item/Tool classes (Pickaxe, Sword, Axe, etc.)
- ItemTier for mining levels
- ItemStack with ItemProperty
- Armor materials
</context>
