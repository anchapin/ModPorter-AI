Create comprehensive input_validator.py with the following:

1. ValidationConfig class:
   - Load and validate YAML configuration
   - Provide default values
   - Support environment variable overrides

2. InputValidationError exception:
   - Custom exception with error code, message, details
   - Support for validation error categories

3. JARValidator class:
   - validate_jar_structure(): Check ZIP/JAR format, required files
   - validate_manifest(): Parse and validate MANIFEST.MF
   - check_jar_size_limits(): Verify extracted size limits
   - detect_suspicious_content(): Scan for malicious patterns
   - Uses zipfile module for JAR handling

4. JavaSourceValidator class:
   - validate_syntax(): Use javalang to parse Java files
   - validate_imports(): Check for valid Java imports
   - detect_forge_mod(): Identify Forge mod patterns
   - detect_fabric_mod(): Identify Fabric mod patterns

5. FileValidator class:
   - validate_extension(): Check allowed file types
   - validate_size(): Check file size limits
   - validate_path_safety(): Prevent path traversal
   - scan_for_malware(): Detect suspicious content

6. InputValidator main class:
   - validate_mod_file(): Main entry point for mod file validation
   - validate_java_source(): Entry point for Java source validation
   - Returns ValidationResult with pass/fail and details
   - Integrates with timeout_manager from 10-01

Key considerations:

- Provide actionable error messages
- Log all validation attempts
- Support batch validation
