# Bedrock Command System
Bedrock uses:
- Minecraft functions (.mcfunction files)
- /execute command for complex logic
- Triggers (/trigger)
- Scoreboard for state management

# Java Command System
Java mods define:
- Command classes with @Command annotations
- CommandSender interfaces
- TabComplete implementations
- Custom argument types
</context>
