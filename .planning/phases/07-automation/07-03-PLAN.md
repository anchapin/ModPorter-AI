**Type**: Implementation
**Effort**: Medium (6 hours)

**Acceptance Criteria**:

- [ ] Mod type classification (tech/magic/adventure/utility)
- [ ] Complexity scoring
- [ ] Dependency analysis

**Implementation**:
```python
def infer_context(mod_features: ModFeatures) -> ConversionContext:
    """Infer conversion context from mod features."""
    context = ConversionContext()
    
    # Infer mod size
    if mod_features.class_count < 10:
        context.mod_size = "small"
    elif mod_features.class_count < 50:
        context.mod_size = "medium"
    elif mod_features.class_count < 100:
        context.mod_size = "large"
    else:
        context.mod_size = "very_large"
    
    # Infer mod type from features
    if mod_features.has_multiblock or mod_features.has_machines:
        context.mod_type = "tech"
    elif mod_features.has_dimension or mod_features.has_biome:
        context.mod_type = "magic"
    elif mod_features.has_entities:
        context.mod_type = "adventure"
    else:
        context.mod_type = "utility"
    
    # Set complexity
    context.mod_complexity = mod_features.complexity_score
    
    return context
```

def get_pattern_defaults(
    mod_features: ModFeatures,
    context: ConversionContext,
) -> Optional[SmartDefaultsResult]:
    """Get defaults from similar successful conversions."""
    
    # Search pattern library
    similar_patterns = pattern_library.find_similar(
        features=mod_features.to_dict(),
        context=context.to_dict(),
        limit=5,
    )
    
    if not similar_patterns:
        return None
    
    # Aggregate successful settings
    successful_settings = [
        p.settings for p in similar_patterns
        if p.success_rate > 0.8
    ]
    
    if not successful_settings:
        return None
    
    # Calculate averaged settings
    result = aggregate_settings(successful_settings)
    result.source = "pattern"
    result.confidence = calculate_pattern_confidence(similar_patterns)
    
    return result
```

**Implementation**:

class UserPreferenceLearner:
    """Learn user preferences from conversion behavior."""
    
    def __init__(self, user_id: str):
        self.user_id = user_id
        self.preferences: Dict[str, Any] = {}
        self.modification_count = 0
        
    def record_modification(
        self,
        default_settings: Dict,
        modified_settings: Dict,
    ):
        """Record when user modifies defaults."""
        
        for key, value in modified_settings.items():
            if default_settings.get(key) != value:
                # Track what user changed
                self._update_preference(key, value)
        
        self.modification_count += 1
        
    def get_preferred_settings(
        self,
        base_settings: Dict,
    ) -> Dict:
        """Apply learned preferences to base settings."""
        
        if self.modification_count < 5:
            return base_settings  # Need minimum examples
        
        # Apply learned preferences
        for key, value in self.preferences.items():
            if key in base_settings:
                base_settings[key] = value
        
        return base_settings
```

### Task 2.5.3.4: Settings Inference Engine

**Type**: Implementation
**Effort**: Medium (8 hours)
**Dependencies**: Tasks 2.5.3.1, 2.5.3.2, 2.5.3.3
**Verification**: Correct settings selected

**Description**: Combine all sources to infer optimal settings.

**Acceptance Criteria**:
- [ ] Priority: User preferences > Pattern > Mode defaults
- [ ] Confidence scoring for all settings
- [ ] Alternatives provided
- [ ] Reasoning logged

**Settings Priority**:
```
1. User Preferences (highest priority)
   - Learned from past modifications
   - Explicit overrides

2. Pattern Library
   - Similar mods with high success rate
   - Same mod type conversions

3. Mode Defaults (fallback)
   - Based on classification mode
   - Standard industry practices
```

**Implementation**:
```python
def infer_smart_defaults(
    mod_path: str,
    user_id: Optional[str] = None,
) -> SmartDefaultsResult:
    """Infer optimal settings from all sources."""
    
    # Step 1: Get mod features and context
    classifier = ModeClassifier()
    features = classifier.feature_extractor.extract_features(mod_path)
    context = infer_context(features)
    
    # Step 2: Try pattern library
    pattern_result = get_pattern_defaults(features, context)
    
    # Step 3: Try user preferences
    user_result = None
    if user_id:
        user_prefs = load_user_preferences(user_id)
        user_result = user_prefs.apply_to_defaults(
            pattern_result.settings if pattern_result else get_mode_defaults(features.mode)
        )
    
    # Step 4: Combine results
    if user_result and user_result.confidence > 0.7:
        final = user_result
        final.source = "user_history"
    elif pattern_result and pattern_result.confidence > 0.6:
        final = pattern_result
    else:
        final = get_mode_defaults(features.mode)
        final.source = "inference"
    
    # Step 5: Generate alternatives
    final.alternatives = generate_alternatives(features, context)
    
    return final
```
