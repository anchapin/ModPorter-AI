# Bedrock UI System
Bedrock uses:
- Custom UI via .json files in ui/ directory
- Screen definitions with controls
- HUD elements and inventory screens
- Form JSON for player interaction

# Java UI System
Java mods define:
- Screen classes (Screen, ContainerScreen, etc.)
- GuiButton, GuiTextField components
- IGuiHandler tile entities
- Container/Inventory classes
