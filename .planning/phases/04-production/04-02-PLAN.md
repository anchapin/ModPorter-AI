## Verification Criteria

### Integration Test

```bash
# Submit conversion job
curl -X POST http://localhost:8080/api/v1/conversions \
  -F "file=@test_mod.jar" \
  -F "options={\"target_version\": \"1.20.0\"}"

# Expected: {"conversion_id": "uuid", "status": "queued"}

# Check status
curl http://localhost:8080/api/v1/conversions/{id}

# Expected: {"status": "processing", "progress": 45, "current_stage": "translating"}

# Get result
curl http://localhost:8080/api/v1/conversions/{id}/download

# Expected: .mcaddon file download
```
