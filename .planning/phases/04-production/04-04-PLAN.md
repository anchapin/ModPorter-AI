## Verification Criteria

### End-to-End Test

```bash
# Upload and convert simple mod
curl -X POST http://localhost:8080/api/v1/conversions \
  -F "file=@simple_item_mod.jar" \
  -H "Authorization: Bearer {token}"

# Wait for completion (poll or WebSocket)
# Download result
curl http://localhost:8080/api/v1/conversions/{id}/download \
  -H "Authorization: Bearer {token}" \
  --output result.mcaddon

# Validate .mcaddon
unzip -l result.mcaddon
# Should contain: manifest.json, behaviors/, resources/
```
