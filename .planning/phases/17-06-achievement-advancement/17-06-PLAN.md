# Bedrock Advancement System
Bedrock uses:
- Advancement files in achievements/ directory
- .json advancement definitions
- Toast notifications for achievements
- Gametest validation

# Java Advancement System
Java mods define:
- Advancement JSON in data/mod/advancements/
- Criteria requirements
- Rewards (items, recipes, loot)
- Parent advancement references
</context>
