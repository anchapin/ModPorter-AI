# Bedrock Rendering System
Bedrock uses:
- Render controller .json files
- Geometry definitions in .json
- Texture references in RP
- Animation controllers
- Entity models via geometry

# Java Rendering System
Java mods define:
- Custom EntityRenderer classes
- Model classes (Biped, Quadruped, etc.)
- Texture mappings
- Animation definitions
</context>
