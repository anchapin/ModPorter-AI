## Implementation Notes

### Context-Aware Translation

The key improvement is moving from syntax-based translation to semantic translation:
1. Parse Java code into full AST with type information
2. Build control flow graph to understand execution
3. Track variable definitions and uses
4. Generate prompts that include semantic context
5. Validate output preserves semantic meaning

### Data Flow Requirements

- Track all variable assignments and modifications
- Handle method calls and return values
- Map Java data types to Bedrock equivalents
- Preserve side effects and state changes
