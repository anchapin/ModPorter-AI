# Bedrock Particle System
Bedrock uses:
- Particle effects in particles/ directory
- Particle definitions in .json files
- minecraft:particle_effect component
- Particle attachments to entities/blocks

# Java Particle System
Java mods define:
- ParticleData/ParticleType registry
- ParticleEmitter classes
- ParticleParamater classes
- Custom particle rendering
