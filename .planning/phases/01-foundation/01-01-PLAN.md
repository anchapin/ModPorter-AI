CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT NOW()
);

    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

**Type**: Implementation  
**Effort**: Medium (3 hours)  
**Dependencies**: None  
**Verification**: GitHub Actions workflow runs on push

**Description**: Create GitHub Actions workflows for automated testing, linting, and Docker builds.

**Acceptance Criteria**:
- [ ] `.github/workflows/ci.yml` created
- [ ] Backend linting (Ruff, Black)
- [ ] Backend testing (pytest)
- [ ] Frontend linting (ESLint)
- [ ] Frontend testing (Vitest)
- [ ] Docker build verification
- [ ] Workflow runs on push to main
- [ ] Workflow runs on PR creation

**Workflow Structure**:
```yaml
# .github/workflows/ci.yml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  backend-lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Set up Python
        uses: actions/setup-python@v5
      - name: Install dependencies
        run: pip install ruff black
      - name: Lint
        run: ruff check backend/ && black --check backend/

  backend-test:
    runs-on: ubuntu-latest
    services:
      postgres: ...
      redis: ...
    steps:
      - uses: actions/checkout@v4
      - name: Run tests
        run: pytest backend/tests/

  frontend-lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Set up Node
        uses: actions/setup-node@v4
      - name: Install dependencies
        run: cd frontend && pnpm install
      - name: Lint
        run: cd frontend && pnpm lint

  frontend-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Run tests
        run: cd frontend && pnpm test
```

**Test Command**:
```bash
# Trigger workflow manually or push to GitHub
gh workflow run ci.yml
```

**Implementation Notes**:
```python
# backend/src/utils/logging_config.py
import structlog

def setup_logging():
    structlog.configure(
        processors=[
            structlog.stdlib.filter_by_level,
            structlog.stdlib.add_logger_name,
            structlog.stdlib.add_log_level,
            structlog.processors.TimeStamper(fmt="iso"),
            structlog.processors.JSONRenderer()
        ],
        wrapper_class=structlog.stdlib.BoundLogger,
        context_class=dict,
        logger_factory=structlog.stdlib.LoggerFactory(),
    )
```
