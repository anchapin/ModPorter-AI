</objective>

<context>
@ai-engine/converters/sound_converter.py (recently created converter pattern)
@ai-engine/knowledge/patterns/ (existing pattern library)

# Bedrock Dimension System
Bedrock uses:
- dimension files in dimensions/ directory
- world template definitions
- biome definitions in biomes_client.json
- feature definitions for world generation

# Java Dimension System
Java mods define:
- DimensionType/Dimension registry
- Biome/Climate settings
- Chunk generator classes
- WorldFeature decorators
- Structure generators (ruins, villages, temples)
</context>
